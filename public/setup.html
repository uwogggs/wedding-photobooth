🔧 Troubleshooting

Camera doesn't work? You MUST use the HTTPS link (port 3004).
HTTP (port 3003) does not allow camera access on phones.


Certificate warning? This is normal — tap Advanced → Proceed.
The self-signed cert is safe for a local wedding network.


QR code shows wrong IP?

  1. Find your IP: open Command Prompt → type ipconfig
  2. Look for Wireless LAN adapter Wi-Fi → IPv4 Address
  3. Type it in the box above and click Update QR
  4. Or restart the server with: HOST_IP=192.168.x.x node server.js

Phone can't connect?

  1. Phone and laptop must be on the same WiFi network
  2. Run this in admin PowerShell to open the firewall:
    netsh advfirewall firewall add rule name="Wedding" dir=in action=allow protocol=TCP localport=3003-3004
  3. Disable Windows Firewall temporarily if still blocked
